# File Converter

React.js app for pasting code from the form converter and creating or previewing files.

## Versions

- **Bootstrap** – Code to paste + Hide/show functionality
- **Version 2-3 Non MVC** – Code to paste + Form validation code to paste + Hide/show functionality
- **Version 3 MVC** – Code to paste + Form validation code to paste + Hide/show functionality (simplified template format)
- **Modern Forms** – Code to paste + Form validation code to paste + Hide/show functionality (Bootstrap, datepicker, state arrays, etc.)

## Setup

```bash
cd file-converter
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `dist/`. Use `npm run preview` to serve the built app locally.

## Tech

- React 18
- Vite 5
